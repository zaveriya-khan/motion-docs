### Motion Docs

A minimal practice project built with React and Tailwind CSS.
It features **draggable cards** created using Framer Motion.

### 🛠️ Tech Stack

React
Tailwind CSS
Framer Motion

### 📦 Installation

git clone https://github.com/zaveriya-khan/motion-docs.git  
cd motion-docs  
npm install  
npm start
